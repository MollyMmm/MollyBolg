Linux 备忘

10.查看本地ip

$ifconfig | grep "inet " | grep -v 127.0.0.1

9.查看磁盘空间

$ df -h

8.显示隐藏文件

$ defaults write com.apple.finder AppleShowAllFiles Yes && killall Finder

1.创建别名

$ cd ~
$ vim .bash_profile
添加 alias ll='ls -al'  #增加相应别名
(如果没有.bash_profile,创建即可)
$ source .bash_profile

2.centos修改hostname

$ vim /etc/hosts
$ vim /etc/hostname
$ hostnamectl set-hostname spider_slave_10
$ hostnamectl status
然后重启服务器

3.lsof

1)查看某端口状态

$ lsof -i:18088

4.netstat

1)查看当前监听端口

$ netstat -a -t --numeric-ports

2)查看某端口监听地址

$ netstat -anp | grep 9000

3)只列出监听中的连接

$ netstat -nlt

SYN_RECV //一个连接请求已经到达，等待确认

ESTABLISHED //正常数据传输状态/当前并发连接数

FIN_WAIT2 //另一边已同意释放

ITMED_WAIT //等待所有分组死掉

CLOSING //两边同时尝试关闭

TIME_WAIT //另一边已初始化一个释放

LAST_ACK //等待所有分组死掉

5.测试端口号

测试端口号当前正常还是处于关闭状态:

telnet IP 端口 或者 telnet 域名 端口

$ telnet bigdata 50070

退出命令:

1)按住control和]
2)输入quit

6.查看war/jar包的内容

$ jar tvf WordCount.jar

7.mac环境下命令的使用

# linux
